Wrap the sidebar with a div class nav-sidebar-wrapper and give it a unque ID.

<div id="nav_sidebar_demo1" class="nav-sidebar-wrapper">

Within the wrapper, you need 2 divs, nav-sidebar and nav-sidebar-overlay :

nav-sidebar contains the menu list nav nav-sidebar-menu. By default it's hidden from view.

<div class="nav-sidebar nav-sidebar-default shadow-depth-1">
    <div class="nav-sidebar-title">
        <a href="#top" class="nav-sidebar-brand">Nav Sidebar</a>
    </div>
    <ul class="nav nav-sidebar-menu">
        <li class="nav-sub-header"><span>Header 1</span></li>
        <li><a href="#">Test 1</a></li>
        <li><a href="#">Test 2</a></li>
        <li class="active"><a href="#">Active 3</a></li>
        <li><a href="#">Test 4</a></li>
        <li class="nav-sub-header"><span>Header 2</span></li>
        <li><a href="#">Test 5</a></li>
        <li><a href="#">Test 6</a></li>
        <li><a href="#">Test 7</a></li>
        <li><a href="#">Test 8</a></li>
    </ul>
</div>

nav-sidebar-overlay is the overlay when the sidebar is shown to prevent main body from scrolling.

<div class="nav-sidebar-overlay" data-toggle="nav-sidebar-toggle" data-target="#nav_sidebar_demo1"></div>

To toggle the sidebar on/off, simply add the

data-toggle="nav-sidebar-toggle"

and

data-target="#your_wrapper_id"

to any element, such as button and overlay. data-target should contain your wrapper's ID.

I had given the nav-sidebar-overlay those properties so that user can click the overlay to close the sidebar.

Full example code:

<div id="nav_sidebar_demo1" class="nav-sidebar-wrapper">
    <div class="nav-sidebar nav-sidebar-default shadow-depth-1">
        <div class="nav-sidebar-title">
            <a href="#top" class="nav-sidebar-brand">Nav Sidebar</a>
        </div>
        <ul class="nav nav-sidebar-menu">
            <li class="nav-sub-header"><span>Header 1</span></li>
            <li><a href="#">Test 1</a></li>
            <li><a href="#">Test 2</a></li>
            <li class="active"><a href="#">Active 3</a></li>
            <li><a href="#">Test 4</a></li>
            <li class="nav-sub-header"><span>Header 2</span></li>
            <li><a href="#">Test 5</a></li>
            <li><a href="#">Test 6</a></li>
            <li><a href="#">Test 7</a></li>
            <li><a href="#">Test 8</a></li>
        </ul>
    </div>
    <div class="nav-sidebar-overlay" data-toggle="nav-sidebar-toggle" data-target="#nav_sidebar_demo1"></div>
</div>
