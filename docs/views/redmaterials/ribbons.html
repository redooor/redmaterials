Add class corner-ribbon-wrapper to the container of the ribbon.

Then add an element with classes corner-ribbon [position] [bg-color] [text-color] shadow

shadow is optional.

Supported positions are:

  • top-left
  • top-right
  • bottom-left
  • bottom-right

Corner ribbons in Jumbotron

Note: You can even use the contexture bg such as .bg-danger for the ribbon too.
