Add class corner-ribbon-wrapper to the container of the ribbon.

Then add an element with classes corner-ribbon [position] [color] shadow

shadow is optional.

Supported positions are:

  • top-left
  • top-right
  • bottom-left
  • bottom-right

Supported colors are:

  • red
  • pink
  • purple
  • deep-purple
  • indigo
  • blue
  • light-blue
  • cyan
  • teal
  • green
  • light-green
  • lime
  • yellow
  • amber
  • orange
  • deep-orange
  • brown
  • grey
  • blue-grey
  • white
  • black
