Ripple effect

Ripple effect requiers JS for creating DOM-element and controlling animation. Animation work on CSS3. If you want to use this effect, you should add class .ripple-effect to your button.

<button type="button" class="btn btn-default btn-raised ripple-effect mb-2">Default</button>
<button type="button" class="btn btn-primary btn-raised ripple-effect mb-2">Primary</button>
