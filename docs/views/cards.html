Card with Table

Cards in Bootstrap v4 do not support table by default. But we can still use utilities to create the same effect.

Featured table
Col 1 Col 2 Col 3
Row 1 Row 1 Row 1
Row 2 Row 2 Row 2
Row 3 Row 3 Row 3 
<div class="card">
    <h5 class="card-header">
        Featured table
    </h5>
    <table class="table table-striped mb-0">
        <thead>
            <tr>
                <th class="border-top-0">Col 1</th>
                <th class="border-top-0">Col 2</th>
                <th class="border-top-0">Col 3</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Row 1</td>
                <td>Row 1</td>
                <td>Row 1</td>
            </tr>
            <tr>
                <td>Row 2</td>
                <td>Row 2</td>
                <td>Row 2</td>
            </tr>
            <tr>
                <td>Row 3</td>
                <td>Row 3</td>
                <td>Row 3</td>
            </tr>
        </tbody>
    </table>
</div>
